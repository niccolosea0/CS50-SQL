# CS50 Introduction to SQL  

This repository contains my notes, problem sets, and practice solutions from Harvard's **CS50: Introduction to Databases with SQL** course.  

## About the Course  
CS50’s Introduction to Databases with SQL is an entry-level course that teaches:  
- How to model real-world entities and relationships using databases.  
- How to design schemas, use normalization, and enforce constraints.  
- How to query databases with `SELECT`, `JOIN`, `GROUP BY`, and more.  
- How to use indexes, transactions, and views for efficiency and safety.  
- How to apply SQL in real-world scenarios.  

## Repository Structure  
- `notes/` – Personal notes from lectures and readings.  
- `problem_sets/` – Solutions to problem sets.  
- `projects/` – Larger projects and assignments.  
- `practice/` – Extra practice queries and exercises.
